请求参数

参数 类型 必填 说明
Authorization header 是 Bearer YOUR_API_KEY
prompt string 是 图片生成提示词
image_size string 否 分辨率：1K / 2K / 4K（默认1K）
aspect_ratio string 否 宽高比（不传则自动适配）：
• 1:1 正方形 (1024×1024)
• 2:3 竖屏照片 (896×1344)
• 3:2 横屏照片 (1344×896)
• 3:4 竖屏标准 (864×1184)
• 4:3 横屏标准 (1184×864)
• 4:5 社交媒体竖屏 (960×1216)
• 5:4 社交媒体横屏 (1216×960)
• 9:16 竖屏手机 (768×1344)
• 16:9 横屏宽屏 (1344×768)
• 21:9 超宽屏 (1536×640)
model string 否 指定模型名称
